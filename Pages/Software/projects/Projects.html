Extremely Minimalistic Text Editor


In short, the name is Simpletext, and it was a project made when I was learning Nodejs. See, there's a module called Electron, which allows me to code html and css and display it as a desktop app. Electron is cross platform, and has easy tools to package and distribute the app. Simpletext allows for quick and easy text manipulation, though it's buggy and unfinished.

Blooket Pin Finder


Pretty much exactly what it sounds like. I was bored so I decided to mess around with the blooket api using the Python requests module. I managed to decode the json status of random blooket pins to determine if any of the pins were actually an online session.
